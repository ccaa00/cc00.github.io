A Disaster Warning Platform on Harmonizing with Crowdsourced Data, Multinational Satellite Images, and Institutional Crisis Information for Big Data Computation

Overview

This project focused on collecting, analyzing, and managing Volunteered Geographic Information (VGI) and Big Data about large-scale disasters and disaster prevention from selected sources like social media, OpenStreetMap (OSM), humanitarian satellite images. Based on this information, we provided meaningful data visualization, and set up a standard operating procedure for rescue teams.

Firstly, we used the computation of big data (HDFS of Hadoop MapReduce) to abstract the data after the granular data are identified, and analyzed the text content by semantic technologies to replace unstructured data with structured data. Then, we modeled and visualized disaster data (SAS Text Miner Solutions) from open data and social media data. Finally, we provided an information integration platform and information validation mechanism (Crowdsourcing) to monitor complex disasters and strengthen the ability of evacuation.
