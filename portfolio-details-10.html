Urban Mappiing Tool

Overview

We presented a mapping application that could easily create and change the information of buildings and roads on OpenStreetMap (OSM). OpenStreetMap is one of the most popular free and open-source maps with over 2 million registered users all over the globe. It aims to create a free editable geographic database of the world, and the geodata underlying the maps is considered the primary output of OpenStreetMap.

Our application allowed users to create and edit objects in OpenStreetMap. It supported creating nodes, lines, and polygons on the map, and editing points of interest features like building types, materials, entrances, levels, and so on. Users could use it both online and offline, and the application would automatically switch by the strength of the signal. When users made changes with offline mode, those changes would be stored in user devices. And those changes would be uploaded to OSM server immediately once users went back to the places with fair signal.
